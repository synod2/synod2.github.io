블로그 테스트 페이지입니다. 

새로운 포스트 추가를 위해서는 _posts 디렉토리에 파일을 추가하면 된다. 
파일 이름 형식은 `YYYY-MM-DD-name-of-post.md`

코드 표시 구문도 사용 가능함 . 

```python

def print_hi(name):

  print("hello", name)
  
print_hi('Tom')

```
